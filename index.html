Absolute Truths in Web Design

We want to make the internet better. The main goal of the internet is for people to consume information and too many websites prevent this through poor typography or not making their sites accessible to every user. Let's fix that.

Measure

Measure is the length of a line of text.

For a single-column design measure should ideally lie between 40 & 80 characters. Many typographers consider the perfect measure to be 65 characters.

Leading

Leading is the distance in between lines of text.

Titles should be set at a line-height of 1.3.

Copy should be set to a line-height of 1.5-1.65.

Text Contrast

Text contrast is the ratio of the luminance of the text color to that of its background.

The minimum contrast between main copy and its background should have a ratio of 4.5:1.

Main Copy Font-Size

Font-size is the size of the font measured in px and rem or em respectively.

Your main copy should be a minimum of 16px.

This is Good

This font is set to 20px, and you can easily read it. If the font-size is smaller than 16px, some people won't be able to read it, whereas if it's set larger than the default, there aren't any downsides: everyone will be able to read it.

This is No Good

This font is set to 14px. The main goal of your website is to get people to consume your content, and it'll be really difficult for them to do so if they can't read your text.

Images should be set to max-width:100%.

Tap Target

On touch-based user interfaces, tap targets are touchable interface elements that users tap with their fingers to invoke a particular action.

Tap targets should measure a minimum of 44 x 44 points width and height. On standard resolution, 1pt = 1px; on retina display, 1pt = 2px.

Tabbed Navigation

Tabbed navigation allows users to navigate through focusable HTML elements with the tab key on the keyboard.

Make sure users can easily tab to and through navigation links. By default it navigates through elements in the order they appear in the raw html.
